¿Para qué utilicé HTML Y CSS?

HTML

En la parte del HTML nos vamos a encontrar completamente con el funcionamiento de nuestra página. Ahí estará escrito todo el código necesario para que desarrolle las funciones que se esperen de la página, y es muy importante que podamos ordenar nuestro proceso y así no perdernos en la marcha.

CSS

Con esta herramienta te vas a encargar de darle forma a todo ese montón de código escrito anteriormente, porque sí, aun sí ya terminaste de escribir el HTML, ese código aún no tiene forma. Te recomiendo que consultes sobre CSS y HTML para así tener una vista más amplía de lo que vas a hacer.

Extensiones que recomiendo para "Visual Studio Code"

Beautify

Esta extensión es de mucha ayuda a la hora de mantener un orden en el HTML y CSS, pues conforme vamos escribiendo barras de código, nuestra interfaz de VSC (Visual Studio Code) se desordena y algunas partes del código quedan fuera de nuestro rango de visión. Su función más usada es ctrl+shift+p: beautify file, la cual ordena el texto de manera instantánea.

VS Code Icons

Esta herramienta, es utilizada para dotar de un símbolo a nuestros archivos. Estos los podemos visualizar cuando creamos un archivo .html o .css en los cuales se podra ver su símbolo individual.

Live Server

La extensión en cuestión, nos sirve para una de las partes más fundamentales a la hora de estar haciendo nuestra página web, la cual es, pre-visualizar el contenido de nuestra página en tiempo real, creando una ventana en nuestro navegador y actualizando los cambios completamente en vivo.
